## 注意事项

### 1. 缓存清理
修改后可能需要清理浏览器缓存或强制刷新页面才能看到更新效果。

### 2. 字体长度
"江苏联储ERP管理系统"比原来的标题更长，需要确认在各种显示场景下不会出现文字截断或布局问题。

### 3. 多语言支持
如果系统后续需要支持多语言，需要将这些硬编码的标题移到语言配置文件中。

### 4. SEO影响
页面标题的修改可能会影响搜索引擎优化，需要相应更新网站的SEO配置。

### 5. Logo文件依赖
修改后的logo引用依赖于`public/logo.png`文件，确保该文件存在且格式正确。建议logo图片：
- 尺寸：32x32像素或更高分辨率的正方形
- 格式：PNG格式，支持透明背景
- 文件大小：建议小于100KB
